Puertos serie

Por defecto la aplicación envía la información a la Raspberry Pi Pico por los puertos serie 'COM3' para Windows y '/dev/ttyACM0' para linux, si estos puertos no funcionaran para tu equipo aquí tienes un listado de los puertos serie que puedes usar para probar. Usa el argumento '--linuxPort=xxx' o '--winPort=xxx' a la hora de ejecutar el .jar
